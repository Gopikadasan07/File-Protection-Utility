Overview

This is a simple Python project that protects data by converting it into a Base64 encoded format and restores it by decoding the protected data.

Features
Protects text using Base64 encoding
Restores the original text using Base64 decoding
Simple command-line interface
Beginner-friendly Python project
Requirements
Python 3.x
How to Run
Clone this repository.
Open the project in Jupyter Notebook or any Python IDE.
Run the Python file.
Choose:
1 to protect data
2 to restore data
Example

Protect Data

Enter your choice: 1
Enter text to protect: Hello World
Protected Data: SGVsbG8gV29ybGQ=

Restore Data

Enter your choice: 2
Enter protected data: SGVsbG8gV29ybGQ=
Restored Data: Hello World
Technologies Used
Python
Base64 Module
